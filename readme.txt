Esta aplicación Farmacia EV está creada para un delivery de productos farmacéuticos. Para poder usarla se necesitan los siguientes pasos:

1.- Entrar a la aplicación
2.- Crear un usuario con un correo existente y una contraseña
2.1.- En caso de tener una cuenta existente, saltar al paso 3
3.- Iniciar sesion con los datos requeridos
4.- Para ver los productos, darle al botón con el ícono de carrito
5.- Elegir los productos
6.- Comprar
7.- Si quiere cerrar sesion, darle al botón con un ícono de persona y darle a la opción "cerrar sesión"
